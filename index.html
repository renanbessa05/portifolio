// 01. Sobre mim

Quem sou eu?

Sou Renan Bessa Abdo, desenvolvedor web com foco em construir aplicações robustas e escaláveis. Minha jornada no desenvolvimento começou com a curiosidade de entender como as coisas funcionam "por baixo dos panos" — e hoje transformo essa curiosidade em soluções reais.

Tenho experiência sólida no ecossistema JavaScript, desde o front-end com React até o back-end com Node.js. Também trabalho com bancos de dados SQL e aplico princípios de Programação Orientada a Objetos para criar código limpo e manutenível.

Sou adepto do Vibe Coding — uma abordagem moderna que combina meu conhecimento técnico com ferramentas de IA (Google Gemini, ChatGPT, Claude e V0 by Vercel) para entregar soluções mais rápidas, criativas e de alta qualidade.
